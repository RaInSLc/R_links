# UI 精简与体检控制优化计划

根据您的反馈，我们将移除冗余的全局版本设置，并增加对异步体检任务的实时“停止”控制，使界面更加简洁高效。

## 方案设计

### 1. UI 布局重绘 (resource.h & app_window.cpp)
- **移除冗余**：彻底删除“全局版本”标签及输入框。
- **按钮精简**：
    - 将“开始全量异步体检”按钮缩短为“开始”。
    - 新增“停止”按钮，用于随时中断正在进行的搜索任务。
- **排版优化**：粘贴、清空、开始、停止四个按钮将并排整齐排列，布局更加紧凑。

### 2. 异步中断逻辑 (app_window.hpp & app_window.cpp)
- **引入原子标志位**：在后台线程中增加 `m_stopRequested` 标志位。
- **平滑退出机制**：在后台搜索的每一个包循环前都会检测该标志位。一旦您点击“停止”，搜索线程将安全退出，不再消耗额外的网络和 API 额度。

## 验证计划
### 场景回测
- **界面观察**：确认“全局版本”已移除，操作排布局合理。
- **逻辑测试**：在开始体检后点击“停止”，观察状态栏提示并确认任务立即终止。
