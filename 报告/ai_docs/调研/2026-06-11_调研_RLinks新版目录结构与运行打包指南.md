## 运行方法

在运行或调试项目前，确保本地已安装有 Node.js (建议 v18+) 和 Rust 开发环境 (包含 Cargo)。

### 1. 安装前端依赖
在 `z:\R_links\mod_UI\` 目录下打开终端，执行以下命令安装 Node 依赖：
```powershell
npm install
```

### 2. 启动本地开发与调试环境
执行以下命令启动开发环境，Tauri 会自动开启前端开发服务器并编译 Rust 后端，最终拉起带有热重载调试框的桌面窗口：
```powershell
npm run tauri dev
```
或直接使用：
```powershell
npx tauri dev
```

## 打包方法

打包会将前端资源进行静态编译（Vite Build），并编译 Rust 后端代码的 Release 版本，最终生成适用于 Windows 的独立安装包（如 `.msi` 或 `.exe` 格式）。

### 1. 执行 Release 打包
在 `z:\R_links\mod_UI\` 目录下打开终端，执行以下命令：
```powershell
npm run tauri build
```
或使用：
```powershell
npx tauri build
```

### 2. 获取打包产物
打包完成后，编译好的安装程序和独立二进制文件将生成在以下目录中：
```
z:\R_links\mod_UI\src-tauri\target\release\bundle\
```
其中：
- **MSI 安装器**：位于 `bundle\msi\` 目录下（例如 `R_Links_0.1.0_x64_zh-CN.msi`）。
- **可执行程序**：独立的 `.exe` 程序位于 `target\release\` 目录下。
