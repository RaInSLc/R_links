# R包搜索及定位优化记录 - 20260502

## 任务目标
解决 RLinkInstaller 无法搜到 `FigR`, `Pando`, `Seqinfo`, `BSgenome.Mmusculus.UCSC.mm10` 等包的问题。

## 诊断发现
1.  **Bioconductor 解析失效**：HTML 标签匹配过于严格（`<td>Version</td>`），且部分包（如 BSgenome 系列）属于 `data/annotation` 分类，之前的代码虽然增加了分类但解析逻辑仍可能因空白符失败。
2.  **GitHub 搜索噪音**：`Pando` 被 `pandoc` 干扰，因为 `pandoc` 包含 `pando` 字符串且星标更高。
3.  **r-universe 数据结构**：`r-universe` API 返回的是数组，且 JSON 解析逻辑对空格敏感。
4.  **环境问题**：编译过程中发现 `RLinkInstaller_PRO.exe` 被占用，导致无法更新。已通过 `move` 命令解除占用。

## 修改计划
1.  **app_window.cpp**:
    - 改进 Bioconductor 版本解析逻辑，使用更宽松的字符串查找。
    - 改进 GitHub 评分系统，完全匹配包名（忽略大小写）获得最高优先级。
    - 强制执行 GitHub `DESCRIPTION` 反向核对，防止误装同名或包含名的无关包。
    - 优化 `r-universe` 解析，自动提取 GitHub 托管地址。
