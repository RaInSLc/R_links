# 新增一键打包脚本说明

> 生成日期: 2026-06-12 | 类型: 总结 | 状态: 已确认

## 背景
用户需要一个简单的一键打包脚本，以便将该 Tauri+React 项目打包为可执行文件（exe），并能够快速找到打包后的产物。

## 核心功能
1. **自动切换目录**：自动切换到脚本所在的相对路径 `mod_UI` 下，保证随处执行。
2. **依赖检查**：自动运行 `npm install` 确保前端与构建依赖被正确安装。
3. **调用 Tauri 构建**：使用 `npm run tauri build` 触发底层的 Vite + Rust 编译，生成 release 版本的独立运行程序。
4. **归档存放**：打包成功后，在项目根目录自动创建 `release` 文件夹，并将构建产物 `src-tauri\target\release\mod_ui.exe` 提取并重命名复制为 `RLinks_UI.exe`，方便用户直接分发使用。
