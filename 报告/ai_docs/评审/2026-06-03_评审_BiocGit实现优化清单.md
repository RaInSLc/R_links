**正确结果**：GSVA 1.50.5 → 应为 Bioconductor 3.18

**正确公式推导**：
- 1.50.x → 3.18
- 1.52.x → 3.19
- 1.54.x → 3.20

公式：`Bioconductor 版本 = 3 + (包次版本号 - 50) / 2 + 18`

简化：`Bioconductor 次版本号 = (包次版本号 - 50) / 2 + 18`

验证：
- (50 - 50) / 2 + 18 = 0 + 18 = 18 ✓
- (52 - 50) / 2 + 18 = 1 + 18 = 19 ✓
- (54 - 50) / 2 + 18 = 2 + 18 = 20 ✓

    ~WinInetRequest() { Close(); }
    
    HINTERNET OpenUrl(const std::wstring& url, const std::wstring& headers = L"") {
        m_hRequest = InternetOpenUrlW(m_hInternet, url.c_str(), 
            headers.empty() ? NULL : headers.c_str(), -1L, 
            INTERNET_FLAG_NO_CACHE_WRITE, 0);

            m_hRequest = NULL;
        }
    }

    operator HINTERNET() const { return m_hRequest; }
    
private:

};
