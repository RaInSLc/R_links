# 启动故障修复报告 - 202604101430

## 1. 故障现象
程序直接双击打不开，显示错误弹窗：
- 标题：错误
- 内容：窗口创建失败！

## 2. 诊断逻辑
通过检查 `z:\pythonProject\R_links\cpp_src\main.cpp` 和 `app_window.cpp`：
- 在 `main.cpp` 中发现报错点位于 `app.Create` 返回 false 处。
- 深入分析 `app_window.cpp`，对比 `RegisterClass` 和 `CreateWindowExW` 两个关键函数。
- 确认 `RegisterClass` 注册的类名为 `L"RLinkInstallerClass"`，而创建窗口时填写的类名误写成了 `L"RPkgInstallerClass"`。

## 3. 修改内容
**文件**: `cpp_src\app_window.cpp`
```cpp
// 修改前
m_hwnd = CreateWindowExW(0, L"RPkgInstallerClass", ...);

// 修改后
m_hwnd = CreateWindowExW(0, L"RLinkInstallerClass", ...);
```
